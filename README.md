# Pluss100 - Landing Page

Landing page para **Pluss100**, software contable 100% en la nube diseñado para PYMES y startups colombianas.

## Descripción

Pluss100 es un centro de mando financiero que incluye:

- Facturación electrónica validada ante la DIAN
- Dashboard gerencial con indicadores en tiempo real
- +15 informes tributarios automáticos (IVA, Retención, Renta, ICA, etc.)
- Gestión de nómina y activos fijos integrada

## Tecnologías

- **HTML5** semántico con SEO optimizado (Open Graph, Twitter Cards, JSON-LD)
- **Tailwind CSS** (CDN con prefijo `tw-`)
- **CSS** personalizado ([css/styles.css](css/styles.css))
- **JavaScript** vanilla ([js/main.js](js/main.js))
- **Google Fonts**: Sora y DM Sans

## Secciones

| Sección | Descripción |

| Testimonios | Reseñas de clientes reales |
| FAQ | 6 preguntas frecuentes con acordeón |
| CTA Final | Llamado a acción para agendar demo |

## Uso

Abrir `index.html` en un navegador. No requiere build ni instalación de dependencias ya que Tailwind CSS se carga desde CDN.
